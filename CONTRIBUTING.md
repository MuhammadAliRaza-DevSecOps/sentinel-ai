# Contributing Guide

## Contribution Kaise Karein

1. **Fork** karo repository
2. **Branch** banao: `git checkout -b feat/amazing-feature`
3. **Code** likho aur tests add karo
4. **Test** karo: `pytest tests/ -v`
5. **Commit** karo: `git commit -m 'feat: add amazing feature'`
6. **Push** karo: `git push origin feat/amazing-feature`
7. **Pull Request** kholo

## Code Style
- PEP 8 follow karo
- Har function pe docstring daalo
- Comments Urdu ya English dono mein okay hain internally
