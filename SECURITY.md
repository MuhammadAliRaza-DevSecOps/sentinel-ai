# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Vulnerability Report Karna
**GitHub Issues mein PUBLIC mat karo security vulnerabilities!**

Email karo: security@yourdomain.com

Include karein:
- Vulnerability description
- Reproduction steps
- Impact assessment
- Fix suggestion (agar ho)

## Response Time
- 24 hours: acknowledgment
- 72 hours: initial assessment
- 7 days: patch/mitigation plan
